##### 프로젝트 목표:

* DirectX 기반의 게임 엔진과 콘텐츠 제작 툴을 통합하여 자체 게임 제작 환경 구축

##### 사용 라이브러리:

Win32API: DirectX 화면 출력용 가벼운 윈도우 창 구현

DirectX11: 2D 이미지 렌더링 구현

MFC: Tool UI 구현, 개발 편의성과 콘텐츠 편집 기능 제공

C++ std::thread: 멀티스레드 데이터 로드 구현으로 리소스 로딩 시간 최소화



##### 프로젝트 폴더 구조 설명:

###### DXEngine

* 게임 개발의 기본 기능과 핵심 시스템을 제공하는 엔진 부분.
* 렌더링, 리소스 관리, 입출력, 멀티 스레드 등 개발 전반에 필요한 기반 기능 포함.

###### DXMiddle

* Client와 Tool 모두에서 공통으로 사용하는 라이브러리와 기능을 통합한 중간 계층.
* 코드 재사용과 모듈화를 위해 설계됨.

###### DXClient

* 최종 사용자에게 보여질 게임 프로그램 부분.
* 게임 콘텐츠와 Client 전용 기능이 포함되어 있으며, 실제 플레이 가능한 결과물 담당.

###### DXTool

* 개발 편의성을 위한 툴 전용 코드.
* MFC 기반 UI 처리, 툴 인터페이스, 콘텐츠 편집 기능 등을 포함.



##### 프로젝트 특징 / 포인트

* 쉐이더를 활용하여 체력 게이지와 UI 요소가 시간에 따라 자연스럽게 변화하는 효과 구현
* 2D 게임 요소별 이미지 출력과 코스튬 적용
* 멀티스레드 데이터 로드를 통한 효율적인 리소스 관리
* Client와 Tool 간 코드 재사용 구조 설계로 유지보수 용이
